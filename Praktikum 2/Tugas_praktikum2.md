**2.**Mengapa sangat penting untuk memahami bahasa pemrograman Dart sebelum kita menggunakan framework Flutter ? Jelaskan!

jawab : Memahami bahasa Dart itu penting sebelum memakai Flutter karena Flutter dibangun menggunakan bahasa Dart. Semua kode di Flutter, mulai dari tampilan aplikasi (UI), logika program, sampai pengelolaan package, ditulis menggunakan Dart.



Jadi kalau belum paham dasar Dart, akan kesulitan menulis dan memahami kode di Flutter. Dengan memahami Dart terlebih dulu, proses membuat dan mengembangkan aplikasi Flutter jadi lebih mudah.



3.Rangkumlah materi dari codelab ini menjadi poin-poin penting yang dapat Anda gunakan untuk membantu proses pengembangan aplikasi mobile menggunakan framework Flutter. 

jawab : 

* Dart adalah bahasa pemrograman yang digunakan oleh Flutter untuk membuat aplikasi mobile, termasuk untuk menulis UI dan logika program.
* Dart memiliki fitur produktif seperti tooling, package manager, dan garbage collection yang membantu proses pengembangan aplikasi.
* Dart bersifat statically typed, sehingga tipe data pada variabel dapat diperiksa oleh compiler untuk mengurangi kesalahan program.
* Dart mendukung konsep Object Oriented Programming (OOP) seperti class, object, inheritance, dan lain-lain.
* Dart menggunakan dua metode kompilasi, yaitu JIT (Just In Time) saat development dan AOT (Ahead Of Time) saat aplikasi dibuild agar performanya lebih cepat.
